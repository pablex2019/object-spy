Object Spy es una herramienta web orientada a QA Automation que inspecciona elementos de cualquier página web y genera automáticamente:

- Selectores CSS
- Localizadores XPath
- Scripts de Playwright

Está diseñada para ayudar a QA Engineers y Automation Testers a identificar localizadores y acelerar la creación de pruebas automatizadas.

Funcionalidades
Analizar cualquier página web pública. Extraer atributos del DOM:
- tag
- id
- name
- type
- text
- placeholder
- className
- Generar selectores CSS
- Generar localizadores XPath
- Generar scripts de Playwright automáticamente
- Copiar selectores y scripts al portapapeles
- Interfaz moderna y responsive construida con Tailwind CSS
